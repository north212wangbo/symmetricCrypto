--How to run---------------------------------------------------------------
To encrypt:
1) Put plaintext in file 'plaintext.txt'
2) In terminal, type "python encrypt.py plaintext.txt [key]", [key] is an 80 bits key, just type 20 hex digits.
3) The ciphertext will be stored in file 'ciphertext.txt'

To decrypt:
1) The ciphertext is in file 'ciphertext.txt'
2) In terminal, type "python decrypt.py ciphertext.txt [key]", [key] should be the same as you choose for encryption.
3) The decrypted message will be stored in file 'decrypted.txt'
